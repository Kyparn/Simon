How standards came about

In the year of 1989 Tim Berners put together a document "Information Management: A Proposal" that laid out his vision for the World Wide Web.
In october 1990 Tim had written the three fundamental technologies that remain the foundation for the Web:
  • HTML: HyperText Markup Language
  • URI: Uniform Resource Identifier (also known as URL)
  • HTTP: HyperText Transfer Protocol
